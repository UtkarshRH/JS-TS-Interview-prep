# Phase 3 Interview Questions

---

## Q1. What is Interface?

**Answer:**

An **interface** defines the **structure of an object** and ensures **consistency across implementations**.  
It acts like a **contract** that objects or classes must follow.

---

## Q2. Interface vs Type?

**Answer:**

Interface is mainly used for **object structure**, while `type` is more **flexible** and can represent:

- unions  
- primitives  
- complex types  

---

## Q3. What are Access Modifiers?

**Answer:**

Access modifiers control the **visibility of class properties and methods**.

- `public` → accessible everywhere  
- `private` → only inside the class  
- `protected` → accessible inside class + child class  

---

## Q4. What is Abstract Class?

**Answer:**

An **abstract class** is a base class that **cannot be instantiated**.  
It can contain **abstract methods** that must be implemented by child classes.
