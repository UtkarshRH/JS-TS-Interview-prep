# 4. First TypeScript Program

## Create a File

```
app.ts
```

---

## Example Code

```typescript
let message: string = "Hello Utkarsh";
console.log(message);
```

---

## Compile TypeScript to JavaScript

```bash
tsc app.ts
```

This generates:

```
app.js
```

---

## Run the Program

```bash
node app.js
```
