# 2. Why TypeScript? (Very Common Interview Question)

## Interview Ready Answer

TypeScript improves **code quality** by adding **static typing**, better **autocompletion**, and **early error detection**.  
It is very helpful in **large applications** where multiple developers work together.

---

## Benefits of TypeScript

- catches bugs early
- better IDE support
- better code readability
- safer refactoring
- widely used in large projects

---

## Companies / Technologies Using TypeScript

- Angular
- NestJS
- Deno
- Many Node.js backends
