# Interview Questions

---

## Q1. What is Type Alias?

**Answer:**

Type alias allows us to create a **custom type** for variables, objects, or functions.  
It improves **code readability** and **reusability**.

---

## Q2. Optional Parameter?

**Answer:**

Optional parameters allow a function parameter to be **omitted** using the `?` symbol.

## Example

```typescript
age?: number;
```

---

## Q3. Rest Parameter?

**Answer:**

Rest parameters allow a function to accept **multiple values as an array**.

## Example

```typescript
function sum(...numbers: number[]) {}
```
